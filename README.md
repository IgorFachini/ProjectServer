# PHP-Slim-Restful
```
Api RESTFULL, trabalhando em conjunto com outros 2 projetos.
App <a href="https://github.com/IgorFachini/Project">Aplicativo</a>.
Site <a href="https://github.com/IgorFachini/ProjectServerManager">Site</a>.

Executando:

(Com php) instalado no path(cmd), coloque o projeto em uma pasta qualquer, navegue pelo cmd dentro da pasta que contem a pasta do projeto, e rode o comando ( php -S localhost:80 ).

(Com Xampp) coloque o projeto em xampp\htdocs, e execute o apache do xampp.

Acessando funçoes do projeto,

Com Postman(Chrome) ou chamadas HTTP, podera realizar requisiçoes para ( http://localhost:80/ProjectServer/api/ ) + Tipo.
ATENCAO: Todas o conteudo de todas as requisicoes devem ser feitas em formato JSON, e retorno de todas as funcoes sera em JSON.
```
